# PowerPoint Table Export Library

`powerpoint-lib` provides utilities to render pandas DataFrames as high-quality images
using the [Great Tables](https://posit-dev.github.io/great-tables/) library and embed
those images into PowerPoint presentations.

## Installation

```bash
pip install -e ./powerpoint-lib
```

## Usage

```python
from pathlib import Path
import pandas as pd
from powerpoint_lib import export_table_to_pptx


df = pd.DataFrame(
    {
        "Region": ["North", "South"],
        "Sales": [374000, 412000],
    }
)

export_table_to_pptx(
    dataframe=df,
    template_path=Path("template.pptx"),
    output_path=Path("output.pptx"),
    font="Source Sans Pro",
    header_fill="#2F5597",
    header_font_color="#FFFFFF",
    row_fill="#FFFFFF",
    row_font_color="#222222",
    alternating_fill="#F2F2F2",
    formatters={"Sales": lambda value: f"${value/1000:.0f}k"},
)
```

The resulting PowerPoint presentation is saved to `output.pptx` with a new slide that
contains a table image generated by Great Tables.
